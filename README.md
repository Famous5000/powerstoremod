# Fam's Power Core mod
A mod that allows you to store ridiculous amounts of power using items. Can be launched.

## Importing

Simply download this repository as a zip, then import it through the `Mods` dialog in Mindustry. Or, unzip this repo inside Mindustry's `mods/` folder.

## Contributing

Feel free to submit ideas to this mod.
